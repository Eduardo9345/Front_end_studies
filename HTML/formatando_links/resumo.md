# Formatação de links

Links tem estados: 
* Visitado
* Não-visitado
* Hover, quando passa o mouse em cima
* Active, quando clica e segura

```
/*Links não visitados, padrão*/
a:link{
    color: #b9c941
}

/*Link visitado*/
a:visited{
    color: #c0c0c0;
}
/*Link quando o cursor está em cima*/
a:hover{
    color: #6d790f;
}
/*Link quando clica e segura*/
a:active{
    color: #e4f371
}
```
